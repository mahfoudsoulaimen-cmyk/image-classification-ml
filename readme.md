# Image Classification using Machine Learning

## Project Overview
This project demonstrates a basic image classification system using Machine Learning in Python. 
The model uses the digits dataset from scikit-learn to classify handwritten numbers.

## Technologies Used
- Python
- Scikit-learn
- Machine Learning
- Support Vector Machine (SVM)

## Result
The trained model can recognize handwritten digits with good accuracy.
